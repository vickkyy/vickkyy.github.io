css之水平导航栏

2017-07-04 • 分类: 技术 • 标签: css

水平导航栏水平居中
方法一： 利用 display: inline-block 和 text-align: center

<div id="main">
  <ul id="nav">
    <li><a>First list</a></li>
    <li><a>Second list</a></li>
    <li><a>Third list</a></li>
    <li><a>Four list</a></li>
  </ul>
</div>

ul {
  list-style-type: none;
}

#main {
  text-align: center;
}

#nav {
  display: inline-block;
  margin: 0;
  padding: 0;
}

#nav li {
  display: inline-block;
  width: 150px;
  height: 30px;
  line-height: 30px;
  border: 1px solid #e7e7e7;
  border-radius: 3px;
  text-decoration: none;
  cursor: pointer;
  background: #e7e7e7;
}

方法二： 利用 float 

<div id="main">
  <ul id="nav">
    <li><a>First list</a></li>
    <li><a>Second list</a></li>
    <li><a>Third list</a></li>
    <li><a>Four list</a></li>
  </ul>
</div>

/*去除默认样式*/
ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
}

#nav {
  float: left;
  position: relative;
  left: 50%;
}

#nav li {
  float: left;
  position: relative;
  right: 50%;
}

#nav a {
  display: inline-block;
  margin: 0 5px;
  width: 150px;
  height: 30px;
  line-height: 30px;
  text-align: center;
  border: 1px solid #eee;
  border-radius: 4px;
  cursor: pointer;
  background: #e7e7e7;
}
