变量作用域

变量作用域都是按嵌套的规则集划分，并且在当前规则集下找不到对应变量时会逐级向上查找，注意这个和原生 CSS 的逻辑是完全不同的。
如果在代码中重写某个已定义变量的值，less变量的计算都会以这个变量最后一次被定义的值为准。这个行为被称为「懒加载（Lazy Loading）」。

@import

针对于模块化开发
@import “foo”; // 若引入less文件，则可省略格式后缀。foo.less被引入
@import “foo.less”; // foo.less被引入
@import “foo.php”; // foo.php会被当做less文件引入
@import “foo.css”; // 被当做原生引入

混合(Mixin)

Less 的混入有两种方式：

  1. 直接在目标位置混入另一个类样式（输出已经确定，无法使用参数）；
  2. 定义一个不输出的样式片段（可以输入参数），在目标位置输出。（注：后面如无特殊说明，mixin 均用来指代此类混入。）
