css动画基础笔记_01

2017-08-17 • 分类: 技术 • 标签: css 动画

转载： https://segmentfault.com/a/1190000007319504

css中动画主要包括两个：1. transition 2. animation

Section1. transition

CSS transition 提供了一种在更改CSS属性时控制动画速度的方法。其可以让属性变化成为一个持续一段时间的过程，而不是立即生效的。

基本语法：

transition 是属性简写，简写语法：

transition: <property> <duration> <timing-function> <delay>
实例

example01:
html:

<div id="test">transition!</div>

css:

#test {
    position: relative;
    transition-property: font-size;
    transition-duration: 4s;
    transition-delay: 2s;
    font-size: 16px;
}
#test:hover {
    transition-property: font-size;
    transition-duration: 4s;
    transition-delay: 2s;
    font-size: 32px;
}

delay 的意义在于指定了动画发生的顺序，使得多个不同的 transition 可以连在一起，呈现复杂的效果。
timing-function 包括：1.ease(默认) 2.linear 3.ease-in 4.ease-out 5.cubic-bezier

example02:
html:

<div class="sidebar">
  <li><a class="menuButton" href="home">Home</a></li>
  <li><a class="menuButton" href="about">About</a></li>
  <li><a class="menuButton" href="contact">Contact Us</a></li>
  <li><a class="menuButton" href="links">Links</a></li>
</div>

css:

li {
  list-style:none;
  margin: 10px 10px;
}
.menuButton {
    display: inline-block;
    width: 80px;
    height: 40px;
    padding: 0 5px;
    transition-property: background, color;
    transition-duration: .5s;
    transition-timing-function: ease-out;
    text-align: center;
    background-color: #248888;
    line-height: 40px;
    color: white;
    font-family: sans-serif;
    font-size: 14px;
    text-decoration: none; 
}
.menuButton:hover {
    transition-property: background, color;
    transition-duration: .5s;
    transition-timing-function: ease-out;
    background: #32B9B5;
    color: black;
    /* h-shadow, v-shadow, blur, spread, color, inset */
    box-shadow: 0 0 1px 1px #61727A;
}

备注：transition的局限

  • transition 需要事件触发，无法在网页加载时自动发生。
  • transition 是一次性的，不能重复发生，除非一再触发。
  • transition 只能定义开始状态和结束状态，不能定义中间状态，即只有两个状态。
  • transition 一条规则，只能定义一个属性的变化，不能涉及多个属性。

Section2 animation

CSS animation 使得可以将从一个CSS样式配置转换到另一个CSS样式配置。动画包括两个部分:描述动画的样式规则和用于指定动画开始、结束以及中间点样式的关键帧。

基本语法：
animation: <name> <duration> <timing-function> <delay> <iteration-count> <direction>;

属性详解：

  • animation-name 规定需要绑定到选择器的 keyframe 名称.
  • animation-duration 规定完成动画所花费的时间，以秒或毫秒计.
  • animation-timing-function 规定动画的速度曲线.
  • animation-delay 规定动画开始之前的延迟.
  • animation-iteration-count 规定动画应该播放的次数.
  • animation-direction 规定是否应该轮流反向播放动画.
实例

example04:
html:

    <p> this is a paragraph! </p>

css:

p {
    animation-duration: 3s;
    animation-name: slide;
}
@keyframes slide {
    from {
        margin-left: 0%;
        background: red;
        width: 100%;
    }
    50% {
        margin-left: 50%;
        background: green;
    }
    to {
        margin-left: 100%;
        background: blue;
        width: 100%;
    }
}
