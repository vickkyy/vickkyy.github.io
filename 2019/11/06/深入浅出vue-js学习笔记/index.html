  • 克隆节点
    • 定义: 将现有节点的属性复制到新节点中,让新创建的节点和被克隆节点的属性保持一致, 从而实现克隆效果.
    • 作用: 优化静态节点和插槽节点(slot node).
    • 区别: 克隆节点与被克隆节点的唯一区别是isCloned属性, 克隆节点的isCloned属性值是true, 被克隆节点的isCloned属性值是false.
